# CS:GO Items App

Este é um aplicativo Android para visualização e gerenciamento de itens do CS:GO.

## Funcionalidades
- Visualização de itens do CS:GO
- Gerenciamento de inventário
- Interface intuitiva

## Estrutura do Projeto
- `app/`: Código-fonte do aplicativo Android
- `demo/`: Arquivos de demonstração
- `build.gradle.kts`, `settings.gradle.kts`: Configuração do projeto Gradle

## Como executar
1. Instale o Android Studio.
2. Clone este repositório.
3. Abra o projeto no Android Studio.
4. Conecte um dispositivo ou use um emulador.
5. Execute o aplicativo.
